
 ANZOpenRailTools

ANZOpenRailTools is a collection of productivity tools designed to enhance rail design workflows in Bentley OpenRail Designer. The plugin provides practical utilities that streamline common engineering tasks, reduce manual effort, and improve design efficiency.

---

## Features

### 1. Regression Line to Civil Geometry

Convert a regression line directly into OpenRail Civil Geometry to accelerate alignment development and design workflows.

#### Overview

Rail survey, track analysis, and data processing workflows often generate regression lines that best represent existing track geometry. Traditionally, converting these regression lines into usable civil geometry requires significant manual effort.

The **Regression Line to Civil Geometry** tool automates this process by transforming regression line elements into fully functional OpenRail civil geometry.

#### Benefits

- Rapid conversion of regression lines into design geometry
- Eliminates repetitive manual geometry creation
- Accelerates existing track modelling workflows

#### Workflow

1. Generate or import a regression line.
2. Launch the **Regression Line to Civil Geometry** tool.
3. Select the regression line element.
4. Continue design development using native OpenRail Designer functionality.

#### Output

The tool creates:

- Civil Geometry complex element based on regression points
- Vertical Profile based on regression points

## Requirements

## Installation

1. Download the latest ANZOpenRailTools release.
2. Move the ANZOpenRailTools.dll to a directory included within your configuration
3. Restart OpenRail Designer.
4. Access the tools through the ANZOpenRailTools keyin.
